## Project introduction

Web-based event announcement platform that connects organizers with participants. Authorized users can create and manage events (title, date, location, category, description). Visitors can browse and search events.

## Current stack

- **Backend:** Spring Boot (JPA, H2, JWT auth, REST API)
- **Frontend:** Static HTML/JS (`frontend/`), served optionally via Spring static resources
- **Features:** Register/login against the database, create events, register for events (authenticated), organizer-only delete

## Purpose

Share event announcements and information in one place.
